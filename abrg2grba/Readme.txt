Png images converted with xem's tool are in AGRB format, HBL expects GRBA format bin files.

This tool inputs a bin file in ABRG and outputs the file in GRBA for use with HBL.

Links:

http://xem.github.io/3DShomebrew/
http://xem.github.io/3DShomebrew/tools/image-to-bin.html

Modded version that allows to ouput AGRB or GRBA:

http://suloku.github.io/image-to-bingrba.html
